# A/B Test Conversion Rates Dashboard

Интерактивная панель управления для визуализации и анализа A/B тестов конверсий.

## Функциональность

### Визуализация данных

- **Линейные графики** с поддержкой различных стилей:
  - Linear (линейный)
  - Smooth (сглаженный)
  - Area (заполненная область)
- **Адаптивный дизайн** - графики подстраиваются под размер экрана
- **Поддержка тем** - светлая и тёмная темы

### Управление данными

- **Множественный выбор вариаций** - возможность сравнивать несколько вариантов одновременно
- **Агрегация по времени**:
  - По дням (детализированный просмотр)
  - По неделям (обобщённый просмотр)
- **Динамическая фильтрация** - мгновенное обновление графика при изменении параметров

### Экспорт данных

- **Экспорт в PNG** - сохранение графиков в высоком качестве
- **Автоматическое именование** файлов с датой экспорта

## Технологии

- **React 18** - пользовательский интерфейс
- **TypeScript** - статическая типизация
- **Recharts** - построение графиков
- **HTML2Canvas** - экспорт в изображение
- **CSS Modules** - стилизация компонентов
- **Vite** - сборка и разработка
